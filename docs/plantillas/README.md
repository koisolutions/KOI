# Plantillas de contratación — KOI (Koi Labs Solutions SpA)

Documentos base para formalizar la relación con un cliente. Están redactados
para el contexto chileno (SpA, SII, Ley N° 19.628 de datos personales) y usan
`[[MARCADORES]]` que debes reemplazar en cada caso.

> ⚠️ **Aviso**: estas son plantillas de referencia, no asesoría legal. Antes de
> usarlas con un cliente real, hazlas revisar por un/a abogado/a en Chile,
> especialmente las cláusulas de propiedad intelectual, responsabilidad y
> tratamiento de datos.

## Opciones según el tipo de proyecto

| Documento | Cuándo usarlo |
|---|---|
| [Contrato de Prestación de Servicios](contrato-prestacion-servicios.md) | Proyecto completo a medida. Es el documento principal y más protector. |
| [Orden de Trabajo (propuesta breve)](orden-de-trabajo.md) | Encargos acotados o clientes recurrentes que ya firmaron un contrato marco. |
| [Acuerdo de Confidencialidad (NDA)](acuerdo-confidencialidad-nda.md) | Antes de conversar detalles sensibles, o cuando el cliente lo exige previo a cotizar. |
| [Welcome Document](welcome-document.md) | Se entrega **después de firmar**, para arrancar el proyecto con todo claro. |

## Flujo recomendado (alineado al proceso KOI)

1. (Opcional) **NDA** si hay información sensible antes de cotizar.
2. **Contrato de Prestación de Servicios** o **Orden de Trabajo** → firma.
3. **Welcome Document** → se sube al portal del cliente.
4. Formulario de onboarding → estrategia → llamada estratégica → desarrollo.
